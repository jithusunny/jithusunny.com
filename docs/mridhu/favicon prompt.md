Create a **minimal monogram favicon** using only a single letter **“M”** for the brand **Mridhu by Elsa**.

### Composition
- Square canvas (1:1)
- The letter **M centered**
- Very tight crop (minimal padding)
- Transparent background (PNG with alpha)

### Typography
- Handwritten calligraphic “M”
- Soft, flowing curves
- Medium-weight strokes (not thin, not bold)
- Feminine, calm, premium feel
- Rounded terminals, no sharp edges

### Graphic Element
- Optional: **very small** botanical accent (single leaf or twig)
- Integrated subtly into one curve of the “M”
- Must not overpower the letter
- Can be omitted if it hurts clarity at small sizes

### Color
- Muted antique gold
  - Hex reference: `#C8A24D`
- Flat color only
- No gradients, shadows, glow, embossing

### Style Constraints
- Flat design
- No background
- No textures
- No mockups
- No text other than the letter “M”

### Output Requirements
- Transparent PNG
- Clean edges
- Readable at very small sizes (16×16)
- Optimized for browser tabs and bookmarks
